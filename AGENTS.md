Before executing any command, log the command into the file `command.txt`.
Log also the errors and warnings generated by the command in `command.txt`. Only log errors and warnings. Don't log the output of the command.
If `command.txt` already exists at the start of a session, make a backup copy and clean the content of `command.txt`.
If `cbuild` fails because of source file path issues, then cancel the build and record the path in `command.txt`.
Don't try to debug the build issue. Stop the process when there is a build issue.
